# 🚂 Тренажер составителя поездов РЖД

Тренажер для обучения составителей поездов правильному ведению переговоров согласно регламенту станции Новосибирск-Восточный.

## 📦 Что нужно для работы

### 1. Скачай модель распознавания речи
- Ссылка: https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip
- Распакуй архив в папку с программой
- Должна появиться папка `vosk-model-small-ru-0.22`

### 2. Установи Python и библиотеки

```bash
# Скачай Python 3.10+ с python.org (обязательно поставь галочку "Add to PATH")
# Открой командную строку и выполни:
pip install sentence-transformers torch rapidfuzz numpy scikit-learn
3. Обучи нейросеть
bash
cd python
python train_model.py
После обучения появится папка trained_models/rzd_sbert_model/

🚀 Запуск
Убедись, что всё на месте:

text
RZDTrainer/
├── RZDTrainer.exe
├── vosk-model-small-ru-0.22/   ← модель Vosk
├── trained_models/
│   └── rzd_sbert_model/        ← нейросеть
├── python/
│   ├── evaluator.py
│   └── train_model.py
├── scenarios.json
└── voice_commands.json
Запусти RZDTrainer.exe

Выбери микрофон

Нажми и держи кнопку "ЗАЖМИТЕ", скажи фразу, отпусти

📋 Сценарии (27 штук)
Все сценарии взяты из официального регламента переговоров станции Новосибирск-Восточный.

СЦ-01: Подтверждение вызова ДСП

СЦ-02: Подтверждение плана маневров

СЦ-03: Запрос маршрута у ДСП

... и ещё 24 сценария

⚙️ Голосовые команды
Команда	Действие
"стой", "стоп"	Аварийная остановка
"дальше", "следующий"	Следующий сценарий
"повтори"	Повтор сценария
"помощь"	Подсказка
🛠️ Сборка из исходников
bash
dotnet restore
dotnet build -c Release
📝 Требования
Windows 10/11 (64-bit)

4 ГБ ОЗУ (рекомендуется 8 ГБ)

Микрофон

2 ГБ свободного места

🐛 Частые проблемы
"Модель Vosk не найдена" → Скачай модель по ссылке и распакуй в папку с программой

"Python не найден" → Установи Python с python.org (галочка "Add to PATH")

Долгая оценка → Первый запуск нейросети дольше, дальше быстрее
