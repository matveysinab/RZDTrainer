# RZDTrainer

Тренажёр регламента переговоров составителя поездов (WPF, .NET 8).

## Что нового
- Обновлённый, более наглядный интерфейс (карточки, тени, бейджи).
- **Сложность сценария** — поле `difficulty` в `scenarios.json`
  ("Лёгкий" / "Средний" / "Сложный"). Показывается в списке сценариев
  и в карточке сценария цветным бейджем.
- **Картинки к сценариям** — поле `image` в `scenarios.json`.
  Сами файлы лежат в папке `images/`.

## Как добавить картинку к сценарию
1. Положите файл (`.png` или `.jpg`) в папку `images/`, например `images/sc-05.png`.
2. В `scenarios.json` у нужного сценария впишите имя файла в поле `image`:
   ```json
   { "id": "СЦ-05", "image": "sc-05.png", ... }
   ```
3. Запустите программу — картинка появится в карточке и миниатюрой в списке.

Подробная инструкция также лежит в `images/КАК_ДОБАВИТЬ_КАРТИНКИ.txt`.

## Распознавание речи (Vosk)
- Все исправления распознанного текста идут через `VoskCorrector` строго по **целым
  словам** — поэтому «иванов» больше не превращается в «иванав», а «поехали» — в «наехали».
- **Привязка к лексике сценариев** включена всегда: незнакомое слово автоматически
  заменяется ближайшим словом из сценариев (если отличие 1–2 буквы). Например
  «сставитель» → «составитель», «мошинист» → «машинист». Это работает на уровне
  результата и не мешает распознаванию.
- **Жёсткая грамматика Vosk** (распознаются только слова из сценариев) по умолчанию
  **выключена**: на маленькой модели `vosk-model-small-ru-0.22` она сильно ухудшает
  распознавание (речь может вообще не распознаваться). Если хотите поэкспериментировать —
  создайте/откройте рядом с программой файл `app_settings.json` и добавьте:
  ```json
  { "UseVoskGrammar": true }
  ```
  Вернуть как было — `false` или удалить строку.
